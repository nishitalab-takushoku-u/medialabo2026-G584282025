数当てゲーム

コンピュータは1から10までの乱数から値を1つ決めて答えとします。
ユーザは答えを予想して入力してもらいます。
ユーザの入力が外れた場合は、ヒントとして予想した数よりも大きいか小さいかを表示します。
3回以内に答えの数と予想した数が合っていればユーザの勝ち、そうでなければユーザの負けです。
3回まちがうと、答えの数が表示されます。
4回以上回答するか正解した後に回答するとゲームが終わっているというメッセージが表示されます。

ユーザはテキストボックス内に予想した数を入力
回答ボタンを押すと判定される

数当てゲームのページ: kazuate.html

各授業回の学習事項

第1回 WWW と HTML

HTMLにおいて<p>や</p>はタグと呼ばれ前者が開始タグ、後者が終了タグと呼ばれる。

タグで囲まれた部分が要素になる。

さらに要素に追加する付加情報として属性がある。
例えば画像を表示したい場合、画像の説明テキストや表示する大きさをそれぞれの属性で指定できる。

演習の成果: 演習1-4: HTMLのさまざまな要素

第2回 CSS による WEB ページの装飾

ページを装飾する記述はCSSと言う。

CSSを追加するには、head要素にlink要素を記述し、別でCSSファイルを作る。

CSSの書式は、セレクタ(装飾したい要素を指定)・プロパティ名(装飾の種類を指定)・プロパティ値(装飾の具体的な値)を記述する。

演習の成果: 演習2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

JavaScriptはWebページ上で複雑な機能を実現するプログラミング言語。

script要素のsrc属性でJavaScriptファイルを指定。
script要素のdefer属性はJavaScriptプログラムの実行を先送りする。

console.log()で文字列を出力する。

for-ofは配列の要素1つずつに対して処理を実行する構文である。

演習の成果: 演習3-6: オブジェクトの参照

第4回 DOM と JavaScript

input要素は入力項目を意味する。
(例)カレンダー・テキストボックス・スライダー・チェックボックス

buttonはボタンの要素である。イベント処理と組み合わせて使う。

要素.getAttribute(属性名)で要素の属性値を参照する。

input要素.valueでユーザが設定した値を参照する。

第5回 イベント処理

createElement(要素名)で要素を新規作成する。

要素1.insertAdjacentElement('beforeend', 要素2)で要素1の内部に要素2を配置できる。
beforeendをafterendにすることで、要素1の直後に要素2を配置できる。

要素.setAttribute(属性名, 値)で要素の属性を設定できる。

要素.remove()で要素を削除できる。

第6回 通信処理

Ajaxはページ全体を再読み込みすることなく、ページの一部を更新する処理。

axios は Ajax 通信を実現するライブラリである。

通信は、次の2つの処理に分けて考えることができる。

  • ブラウザからサーバへの通信: データを要求する（リクエストする）
  • サーバからブラウザへの通信：データ（レスポンス）を返す

2つの通信はインターネットを介して行われる。
ブラウザがリクエストを送ってからレスポンスを受け取るまでには時間がかかる。
通信処理を実行するプログラムには、リクエストしてからレスポンスが返ってくるまで待つ方式と 待たない方式の2通りがある。

  • 同期通信: リクエストを送った後に、レスポンスが返ってくるまで待つ
  • 非同期通信: リクエストを送った後は，ブラウザの別の処理を実行する。 レスポンスの受け取りはイベント処理として記述する

Ajax 通信は、後者の非同期通信方式である。
